# Porque usar o Linux?

O linux oferece um nivel de controle muito mais alto por meio de alguns metodos diferentes. Diferente do Windows Linux é open source, isso significa que o código fonte do sistema operacional é disponivel para voce. Voce pode alterar e manipular como quiser, se voce esta tentando fazer com que o sistema opere de uma forma que nao foi planejada, é essencial poder manipular o codigo-fonte.

Para hackear de forma eficaz, voce deve sabe e entender seu SO e, em grande parte, o sistema que voce esta invadindo. Linux é totalmente transparente essa é uma das grandes vantagens, isso significa que voce pode ver e manipulat todas as suas partes funcionais. Microsoft tenta sempre dificultar o que for possivel para que voce nao tenha conhecimento interior, entao nunca vamos saber o que realmente acontece "sob o capô"

A maioria das ferramentas foram escrita no Linux, 90% podemos dizer é claro que existe outros como Cain e Abel e Wikto. Mesmo quando ferramentas de hacking, como o Metasploit ou o nmap, são portadas para o Windows, nem todos os recursos são transferidos do Linux.

O futuro pertence ao Linux/Unix, Desde que a internet iniciou, Linux/Unix tem sido seu SO como Web Server, muitos roteadores, switches e outros sistemas quase todos usam um Linux Kernel, no mundo da Virtualização é dominada por Linux, Vmware e Citrix construidos sobre o Linux Kernel. Mais de 80% dos mobiles rodam Unix ou Linux, iOS é Unix, e Android é Linux, enquanto MS tem somente 7% de mobiles no mercado.
